# Yahatl.Api.Client

This project will contain the generated TypeScript client for the YAHATL API.

## Purpose

To provide a type-safe client library for consuming the YAHATL API from React Native and other TypeScript/JavaScript applications.

## Contents (to be implemented)

- NSwag-generated TypeScript client
- npm package configuration
- Type definitions for all DTOs
- Fetch API-based HTTP client

## Usage

This client will be generated from the API's OpenAPI specification using NSwag. The generated files will be placed in the `generated/` directory.

The client will be packaged as an npm package for consumption by the React Native mobile application.
